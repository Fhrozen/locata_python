# LOCATA Code for python

This scripts allows to test LOCATA in python (instead of MATLAB)

## Requiriments

- Python 3.6.1
- numpy 1.16.1

## Installation

Install python libraries with miniconda:

```
$ cd tools
$ make
```

## Algorithms

Currently, the code only supports MUSIC.
